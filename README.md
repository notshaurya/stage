## Run Locally

Clone the project

```bash
  git clone https://github.com/notshaurya/stage
```

Go to the project directory

```bash
  cd stage
```

Install dependencies

```bash
  npm install
```

Start the dev server

```bash
  npm run dev
```
Start the production server

```bash
  npm run build
  npm run start
```
